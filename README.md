## 🌟 Overview

Viewix is a premium, next-generation document viewer and workspace environment built with React and Tailwind CSS. It is designed to provide an unparalleled user experience for managing, viewing, and annotating documents right in your browser. 

The application utilizes Framer Motion to provide buttery-smooth micro-interactions and transitions, ensuring that the platform feels responsive, native, and incredibly polished.

## ✨ Key Features

- **🎨 Premium UI/UX:** Stunning glassmorphism effects, meticulously crafted dark and light modes, and seamless page transitions.
- **📚 Template Gallery:** Jumpstart your work with a suite of beautifully designed templates (Invoices, Resumes, Cover Letters, etc.), featuring robust filtering, starred items, and "copy-to-workspace" functionality.
- **📂 Intelligent Workspace:** Dynamic dashboard with Recent Files integration, Starred documents, and real-time state retention.
- **⚡ Power User Ready:** Fully featured Keyboard Shortcuts pane with an official, automatically generated PDF-styled print layout (`Ctrl+P`).
- **🚀 Blazing Fast:** Built on top of Vite and React 18, ensuring lightning-fast HMR and optimized production builds.

## 🛠️ Technology Stack

- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🚀 Getting Started

To run Viewix locally on your machine, follow these steps:

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hd-mixture/Viewix.git
   cd Viewix
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## 🖨️ Professional Print Layouts

Viewix is optimized not just for the screen, but for the page. Try navigating to the **Keyboard Shortcuts** tab and hitting `Ctrl + P`. You'll experience a custom-built `@media print` layout that strips away the web UI and generates a beautiful, A4-optimized, typeset PDF reference guide on the fly.
